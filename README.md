# 🤖 AI Learning Buddy

An AI-powered learning assistant built using **Python, Streamlit, and Google's Gemini API**. This application helps students understand concepts, learn interactively, generate quizzes, receive feedback, and improve their knowledge through AI-assisted learning.

---

## 📌 Features

- 📖 Explain concepts in simple and easy-to-understand language
- 🌍 Provide real-life examples for better understanding
- 📝 Generate multiple-choice quizzes
- 💡 Offer learning feedback and suggestions
- 👨‍🏫 Interactive Full Tutor Session
- ❓ Ask Anything mode for general learning questions
- ⚡ Fast AI-powered responses using Google Gemini

## ⚙️ Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/shanmu-02/AI_PROJECT.git
cd AI_PROJECT
```

### Step 2: Install the Required Packages

```bash
pip install -r requirements.txt
```

### Step 3: Set Your Gemini API Key

Create an environment variable:

```text
GEMINI_API_KEY=YOUR_API_KEY
```

Replace `YOUR_API_KEY` with your Google Gemini API key.

### Step 4: Run the Application

```bash
streamlit run app.py
```

## 🎯 How to Use

1. Launch the application.
2. Enter any topic you want to learn.
3. Select a learning activity.
4. Click the **Generate** button.
5. The AI generates a personalized response based on your selection.
